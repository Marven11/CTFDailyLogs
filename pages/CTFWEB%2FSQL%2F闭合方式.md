title:: CTFWEB/SQL/闭合方式

- 字符串可以没有包裹
- | 数值型             | mysql | mssql | oracle | PostgreSQL |
  | ------------------ | ----- | ----- | ------ | ---------- |
  | ''                 | ✔     | ✔     | ✔      | ✔          |
  | ""                 | ✔     | ✘     | ✘      | ✘          |
  | ()                 | ✔     | ✔     | ✔      | ✔          |
  | ('')               | ✔     | ✔     | ✔      | ✔          |
  | (((((((('')))))))) | ✔     | ✔     | ✔      | ✔          |
  | ("")               | ✔     | ✘     | ✘      | ✘          |
  | (((((((("")))))))) | ✔     | ✘     | ✘      | ✘          |
